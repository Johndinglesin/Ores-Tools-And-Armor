# Ores-Tools-And-Armor
I’ve been working super hard on this new Minecraft mod using MCreator, It’s still in the early stages, but I think its ready to be shared

Right now, I’ve added 3 new materials to the game.

What’s in the Mod?

Here are the three types of materials you can find right now:

Steel: Way more durable than iron. It’s basically the middle ground before you get to the really good stuff.

Ruby: Super rare and looks awesome. The sword does way more damage than diamond!

Bronze: You can find this pretty easily, and it’s great for starting out your world if you can't find iron right away.

Every material comes with:

an Ore

a Block

an Axe

a Pickaxe

and a Sword
